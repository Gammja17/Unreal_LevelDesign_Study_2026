# Unreal Level Design Study 2026

언리얼 레벨 디자인 스터디 기록
강의를 수강하며 **Unreal Engine 5**를 이용한 **레벨 디자인 및 맵 구현 방법**을 학습합니다.
<br><br>

### 스터디 개요

| 항목 | 내용 |
|------|------|
| 강의명 | 레벨 디자인, 기초부터 서류 프리패스 포트폴리오까지 (콜로소, 이용태) |
| 학습 기간 | 2026.03.16 ~ 2026.05.24 (예정) |
| 학습 목표 | - Unreal Engine 5 기반 레벨 디자인 기초 이해 <br> - 플레이 경험 중심의 공간 설계 능력 습득 <br> - 다양한 유형의 레벨 구현 경험 <br> - 포트폴리오 제작 |
